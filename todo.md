 - [x] 贪婪模式匹配任意扩展名

 - [ ] 模糊查找jpeg/jpg图片

## Bugs
### General
 - [x] 如果当前目录无图片，应该弹窗报错/继续搜索
 - [x] 选择关闭翻译之后，文件状态被设为已更改，之后加载的文件也未重置此状态
 - [x] 打开首选项窗口时，状态栏提示快捷键已更新
 - [x] 如果有一个标签，拖动它，然后松开鼠标，再左键新建一个标签，那么前一个标签的位置没有保存，会被重置回其拖动前位置
 - [x] 编辑模式下切换聚焦标签时，光标显示在文本最开头，然而实际位置在文本末尾
 - [x] config中快捷键的"+" 被以 raw unicode 保存
 - [x] "选中标签后自动聚焦到文本框" 关闭状态时，如果原来没有聚焦，那么功能正常。如果原来已经聚焦了文本框，那么切换标记时不会取消焦点，文本框仍持有焦点且可输入，但观察到光标没有出现在内容末尾。可能是只跳过了部分函数调用？
 - [ ] 当已经打开了文件时，创建新翻译文件，创建成功后旧图片与标记未被刷新。此时切换一下图像，可以恢复。
